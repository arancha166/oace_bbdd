Tipo de datos

Ya hemos visto lo que es una base de datos. Está compuesta por un conjunto de tablas que se relacionan entre sí. Dichas tablas están ordenadas por filas y columnas, de forma similar a una hoja de cálculo. En cada una de las filas aparece la información de un registro y cada registro contiene una serie de datos relacionados. Los registros son un conjunto de campos que contienen datos de los mismos tipos que los demás registros. A cada columna de una tabla hay que asignarle el tipo de dato que va a almacenar.  Base tiene hasta 21 tipos de datos distintos, los mas frecuentes son: Numérico(Numeric), Texto fijo(Char), Texto Variable(VarChar)...
