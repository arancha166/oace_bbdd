UD7.BasesDeDatos

Libre Office Base

  • Es una herramienta software integrada en la suite ofimática de LibreOffice  con licencia libre (puedes modificar y redistribuir dicho software)
  • Base es una interfaz de base de datos de datos de escritorio con todas las funciones, diseñado para satisfacer las necesidades de una amplia gama de usuarios. Es mucho más que una herramienta que puede ser usada para fines autónomos, sino que también se integra perfectamente con otras aplicaciones de la suite LibreOffice.
  • Base permite realizar operaciones básicas sobre una BD como abrir, cerrar, guardar, crear, etc. y crear automáticamente BBDD completas (con
    ayuda de plantillas).
