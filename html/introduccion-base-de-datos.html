UD7.BasesDeDatos

Introducción Base de datos

Una Base de datos  es una recopilación de información en forma de uno o varios archivos organizados y vinculados entre sí.

Para explotar las BBDD (bases de datos) se utilizan los Sistemas Gestores de Bases de Datos(SGBD), como MS Access o LibreOffice Base.
