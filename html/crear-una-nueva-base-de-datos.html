UD7.BasesDeDatos

Crear una nueva Base de datos

            Para crear una BD de forma sencilla en Base sigue estos pasos:

             1) Ejecuta LibreOffice Base. Se abrirá el Asistente.

             2) Escoger "Crear una base de datos nueva".

             3) Escoge el motor "HSQLDB" integrado.

            4) Escoge no registrar la BD, pero sí abrirla para su edición.
         

           NOTA: si tienes algún problema puedes que tengas que actualizar el "JRE" en Herramientas>Opciones>Avanzado
